# Smart Tourism Recommendation Dashboard

A full-stack MERN web application inspired by the research topic **“Research and Implementation of Machine Learning Algorithms in Tourism Recommendation.”**

## Stack
- Frontend: React, Bootstrap, Chart.js, Framer Motion
- Backend: Node.js, Express.js
- Database: MongoDB + Mongoose
- Authentication: JWT + bcrypt hashing

## Features
- Landing page with hero banner, featured destinations, search/filter, CTA, footer social links
- User registration/login/logout with preference-based profile fields
- User dashboard with:
  - ML-inspired recommendations
  - Saved destinations
  - Profile summary
  - Analytics charts (categories, top rated, activity)
- Destination details with gallery, ratings, reviews, tips, nearby hotels
- Review/rating CRUD
- Favorites save/remove
- Admin dashboard with:
  - Destination CRUD
  - User management
  - Analytics (users, destinations, reviews, top recommended)
- Contact form storing messages in MongoDB
- Responsive UI, dark mode toggle, hover effects, loading states, pagination

## Setup

1. Create MongoDB database and update `server/.env`:
   - `MONGO_URI`
   - `JWT_SECRET`
   - `PORT` (optional)

2. Install dependencies
```bash
npm install
npm install --prefix server
npm install --prefix client
```

3. Seed sample data (destinations + admin)
```bash
npm run seed --prefix server
```

4. Run both apps
```bash
npm run dev
```

- Client: `http://localhost:5173`
- Server: `http://localhost:5000`
